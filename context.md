# ANTIGRAVITY CONTEXT — KAYLEE NAIL SPA (RICHMOND, KY, USA)

## PROJECT GOAL

Design and build an ultra‑premium, conversion‑focused website for **Kaylee Nail Spa**, positioned as the most luxurious and trusted nail destination in Richmond, KY. The website must justify **high‑ticket pricing ($5,000–$10,000)** by directly increasing bookings, perceived brand value, and average customer spend.

This site should feel like a **boutique luxury beauty brand**, not a local nail shop.

## BOOKING SYSTEM (CORE REVENUE ENGINE)

### Booking Philosophy

Booking must feel like **reserving a luxury experience**, not scheduling a service.

### Required Booking Features

* Embedded booking modal (no page reloads)
* Service-based booking (Manicure / Pedicure / Gel / Fill-ins)
* Optional technician preference
* Time-slot availability
* Mobile-first interaction
* Clear confirmation screen

### Premium Enhancements

* Soft progress animation between steps
* Calm microcopy (e.g., “Almost there…”)
* Optional deposit mention (even if not enforced initially)
* Automatic confirmation message copy

**CTA Language:**

* “Reserve Your Spot” (not “Book Now”)
* “Choose Your Experience” (not “Select Service”)

---

## ANIMATION & MOTION DESIGN (LUXURY SIGNAL)

Animations must be:

* Slow
* Purposeful
* Subtle

### Approved Animations

* Fade + slide on section entry
* Parallax on hero imagery
* Gentle hover lift on cards
* Smooth easing (no sharp motion)

### Strict Rules

❌ No bounce animations
❌ No flashy effects
❌ No fast transitions

Luxury = restraint.

---

## MOBILE EXPERIENCE (NON-NEGOTIABLE)

* Gallery optimized for swipe

The mobile experience should feel **better than desktop**.
